Series Spec — OPPO-style W1 → W8

8 weekly Reels using the same OPPO-style 5-phase structure. Each W has its own data-thematic Phase 1 reveal; everything else is templated.

WNumberPhase 1 mechanism
W1−5.8ppTwo curves diverge — portfolio (solid) below, S&P (dashed) above. Gap is the alpha.
W238%5×10 dot grid wipes in. 19 white (winners) on left, 31 muted (losers) on right.

W4−15.5pp4 yearly bars grow from baseline. 2024 (white) towers below; others muted gray.

Templated phases (identical across all 8)

  • Phase 2 (number tick-in): 2.5–2.75s, scale 0.97 → 1, opacity 0 → 1, linear 200ms.
  • Phase 3 (clean state): 5.0–8.0s. Eyebrow + Meta fade in at 5.0–5.3s, hold 2.4s.
  • Phase 4 (callout): 8.5–10.0s. Number scales to 0.45, translates up. Callout fades in with bold word.
  • Phase 5 (end card): 10.0–11.0s. URL + handle.

Next step — automation pipeline

Once design locked, automate with:

  • Playwright opens each reel, waits for animation, captures PNG frames at 30fps for 11s = 330 frames per reel.
  • FFmpeg compiles PNG sequence → MP4 (H.264, 30fps, 8 Mbps).
  • Single Python script: python generate.py --week 1 → outputs W01.mp4.
  • 8 reels × 5 min per reel = ~40 min for full series export. Or batch: python generate.py --all.

Auto-play behavior

Each reel uses Intersection Observer — when scrolled into view (50% visible), animation plays. When scrolled out, animation resets. ↺ Replay button restarts the active reel.
